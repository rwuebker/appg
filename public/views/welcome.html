Welcome

This web application is a project to experiment with an Apigee data store. Here is a link to more information about Apigee.

Apigee offers users a full back end suite of products to take the burden off of companies and enhance productivity.


Post Items For Sale

This link will check for a user token from Apigee, if none found the user will be asked to log in or sign up. Once the user is authenticated they are able to post items to a mock craigslist type market.


Search Posts For Sale

This link will connect to another collection at the Apigee data store to retrieve items that are posted.
